    The post Project Files first appeared on Perfex CRM.]]>     Upload project file

    Project files is dedicated area for a specific project where files/documents related to the project can be uploaded.

    When you open a project from the top tabs you will be able to see a tab called Files

    Click on the tab called files and you can easily drag and drop the file you need to upload for project.

    When uploading project file the customer contacts which have permission for project will receive an email notification that new project file is uploaded. There is few exception where the contact won’t receive an email nofication:

    • If the switchbox Visible to customer is unchecked before you drag/select the file
    • If the email template located in Setup->Email templates-> New Project File Uploaded (Sent to Customer Contacts) is disabled.

    File discussion/comments

    Each uploaded file for project will have the ability to add comments and discuss various conversations.

    When the project file is uploaded will be shown in the table data and you can click directly on the file name, after clicking the file name a new modal will be shown including the preview of the file (if available) where you can add comments on the right side from the modal.
