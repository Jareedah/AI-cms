Version 1.0.8

  • Changes made in the main perfex theme
  • Added ability to create your own view files without changing the core view files. ex. my_home.php.
  • Added option to copy invoice.
  • Added option to convert estimate/invoice total to words (experimental)
  • Added option to choose does to show column for item taxes in the table.
  • Added ability to add multiple taxes on invoice/estimate item.
  • Added new report for invoices.
  • Added spanish translation.
  • All notifications and activity log for invoice/leads/estimates are translatable.
  • Updated all plugins.
  • Updated CodeIgniter to 3.0.6.
  • Improved tasks sorting.
  • Improved – Item tax name will be shown in the column.
  • Moved all plugins into assets/plugins folder – bower_components no longer used.
  • Removed permission manage mail lists – now mail lists are merged with surveys
  • Removed top navigation time.
  • Fixed problem with dateformat (dd.mm.yyyy) on invoice.
  • Fixed todo items pagination problem.
  • Fixed broken titles links when trying to edit in expenses,contracts and departments.
  • Fixed kb when disabled no showing in admin area.
  • Fixed expense note not showing while edit the expense.
  • Fixed permissions for customer when registering was not giving any permissions.
  • Small bugs fixed.
  • Code improvements.
